# MountBuilder

MountBuilder is a simple tool that helps pastors, preachers, and ministry leaders write and organize sermons. You can build your sermon section by section, add Bible verses directly from an online Bible, and save your work to come back to later.

---

## What You Can Do

- Look up Bible verses and insert them into your sermon
- Fill out a full sermon layout: title, theme, main points, illustrations, and conclusion
- Save your sermon so you don’t lose it if you close your browser
- Make sure your scripture references are typed in correctly
- Use it on your phone or your computer

---

## How It Was Built

This app uses basic web tools: HTML, CSS, and JavaScript. It connects to a Bible API to fetch verses and uses localStorage to save your sermons in the browser.

---

## How to Use It

1. Download the files or clone the project from GitHub
2. Open the file named `index.html` in your browser
3. Type in your sermon details
4. Use the "Insert Scripture" button to add Bible verses
5. Click "Save Sermon" to store your work

> You may need to sign up for a free API key to use the Bible API. Instructions for that will be in the code or another file.
